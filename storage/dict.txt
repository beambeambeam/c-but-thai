] เครื่องหมายวงเล็บเปิด-ปิด
( เครื่องหมายวงเล็บปีกกาเปิด-ปิด
) เครื่องหมายลูกน้ำ
{ เครื่องหมายจุด
} เครื่องหมายเซมิโคลอน
() โดยทั่วไป
, กับ
. แห่ง
; ตั้งแต่บัดนี้เป็นต้นไป
int ตัวแปรประเภทจำนวนเต็ม
float ตัวแปรประเภททศนิยม
char ตัวแปรประเภทตัวอักษร
long ประเภทจำนวนเต็มยาว
short ประเภทจำนวนเต็มสั้น
double ประเภททศนิยมสองเท่า
signed ประเภทจำนวนเต็มแบบมีเครื่องหมาย
unsigned ประเภทจำนวนเต็มแบบไม่มีเครื่องหมาย
struct โครงสร้าง
void ประเภทว่าง
